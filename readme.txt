== Description ==

New Google+1 button allows your blog to be rated. The button also provides a current count of how many times your story has been liked on Google Search Engine. 

Google has just released this new +1 button and you might have seen this on http://Mashable.com as they were one of the lucky Beta testers of Google +1. It works well with New Facebook Share Plugin and TweetMeMe button. All three plugins works great together.

Tested upto 3.1.3


#### Functionality

* New - Google +1 button which shows the number of times your post has been liked on Google Search Engine. Integrates the ability to rate your post with one click.


#### Integration

* Position it Before, After, Before/After on post or page. 

* Use custom CSS on the button code.

* Ability to choose whether to display on post and/or pages. 

* Ability to choose whether or not to SHOW COUNT.

* Works well with New Facebook Share and Tweetmeme Button.


== Installation ==

Follow the steps below to install the plugin.

1. Upload the plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Google +1" option to configure the button
